Download

There are different options for downloading.

  • The easiest is to install through Plugin Manager.
  • There's a full download, which contains the help file, and almost all the standard Python libraries (with "_FULL" in the name)
  • ... and there's a minimised version, which doesn't contain the help file, and only includes the basic Python libraries. You should only take this one if you're concerned about space (it's an extra couple of meg)

There's no difference between the zip and the 7zip versions, just that the 7zip versions are smaller.
